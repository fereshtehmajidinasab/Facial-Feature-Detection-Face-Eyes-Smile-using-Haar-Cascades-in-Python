# Facial Feature Detection (Face, Eyes & Smile) 

Hi there!  

This project is a simple example of facial feature detection using **Python** and **OpenCV**. It detects **faces, eyes, and smiles** in real-time using Haar Cascade classifiers.

Itâ€™s a great beginner-friendly computer vision project if you want to understand how object detection works with a webcam.

---

##  What This Project Does

When you run the program:

- It opens your webcam.
- Detects faces in the video stream.
- Detects eyes inside each detected face.
- Detects smiles inside the face area.
- Draws rectangles around the detected features in real time.

Everything happens live using your camera.
